# Source Map

Use this file to plan search. Prioritize fresh, source-backed items that can become Chinese short-video topics.

## Primary AI Sources

- OpenAI: official blog, product docs, X announcements, GitHub.
- Anthropic: news, Claude updates, model cards, research.
- Google DeepMind / Google AI: Gemini, Veo, Genie, world models, developer updates.
- Meta AI: Llama, SAM, Segment Anything, research releases.
- Microsoft AI / GitHub: Copilot, Azure AI, GitHub repos, developer workflows.
- NVIDIA: Omniverse, ACE, robotics, game AI, GPU/cloud announcements.
- Hugging Face: trending models, spaces, papers, community demos.
- Mistral, xAI, Perplexity, Runway, Pika, Luma, ElevenLabs, Suno, Midjourney, Cursor, Replit, Lovable, Windsurf, Cognition, Devin, Anysphere.

## AI Game Development Sources

- Unity blog and Unity AI pages.
- Unreal Engine / Epic developer posts.
- Game Developer, 80 Level, GamesBeat, GameFromScratch.
- NVIDIA ACE, Omniverse, Audio2Face, DLSS/AI game tooling.
- Roblox Creator Hub and AI creator tool updates.
- Scenario, Meshy, Kaedim, Luma, Runway, Tripo, CSM, Convai, Inworld, Rosebud, Leonardo, Stability, World Labs, Odyssey, Genie-style world model projects.
- GDC talks and demos when they are recent or newly released.

## AI Monetization And Builder Sources

- Product Hunt launches.
- Hacker News discussion on AI products and developer tools.
- GitHub Trending and fast-growing repos.
- Indie Hackers, Y Combinator, a16z, Sequoia, Latent Space, Lenny's Newsletter when relevant.
- Founder posts on X/LinkedIn with metrics, launches, waitlists, revenue, pricing, or workflow proof.

## News And Analysis Sources

- The Verge, TechCrunch, VentureBeat, MIT Technology Review, Wired, Bloomberg, Reuters, CNBC, The Information, Semafor.
- Ben's Bites, The Rundown AI, Import AI, Latent Space, Simon Willison, One Useful Thing.
- Use media sources for context, but prefer official/product sources for factual claims.

## Search Query Patterns

Use combinations like:

- `AI game development world generation latest`
- `AI 3D world model game development`
- `AI NPC game development tool`
- `Unity AI game development announcement`
- `Unreal Engine AI tool`
- `AI agent developer tool launch`
- `AI coding tool update`
- `AI app builder launch`
- `AI workflow automation revenue`
- `Vibe coding monetization AI`
- `site:github.com AI game dev`
- `site:news.ycombinator.com AI coding tool`
- `site:producthunt.com AI game development`

For Chinese output, keep English source names but translate the short-video angle into Chinese.
