# Plain-Language Rewrites

Use this reference when writing or revising short-video scripts. The goal is not to remove all technical terms. The goal is to translate each technical term into a concrete person, scene, action, and result.

## Rewrite Table

| Abstract phrasing | Plain-spoken rewrite |
|---|---|
| 可交付结果 | 做出一个客户明天就能打开用的 AI 工具 |
| 企业 AI 落地 | 让公司员工真的在每天的工作里用上 AI |
| AI 工作流 | 把查资料、写文案、填表、发消息这些步骤串起来，让 AI 一步步帮人干 |
| 赋能企业 | 帮公司把重复活拆给 AI 干 |
| 实现增长 | 帮公司多拿客户、多卖货、少流失 |
| AI 解决方案 | 一套能帮某个岗位省时间、少犯错、或者多赚钱的 AI 用法 |
| 行业机会 | 普通人可以切进去赚钱、涨粉、做服务的位置 |
| 平台生态 | 一群公司、开发者、顾问围着这个平台做产品和服务 |
| 商业化路径 | 你准备卖给谁、帮他解决什么问题、他为什么愿意付钱 |
| 提升效率 | 原来要几个小时的重复活，现在让 AI 先做一版，人只负责检查和改 |
| 方案交付 | 不是讲课讲完就走，而是把账号、表格、机器人、自动化流程都配好 |
| AI 顾问 | 帮老板看清哪些活该交给 AI、怎么接进现有流程、员工怎么用起来的人 |

## Sentence Checks

- If two consecutive sentences are abstract judgments, rewrite one into a scene.
- If a technical term appears without an example, add one.
- If viewers cannot picture the scene, add a role, tool, task, and result.
- If the sentence could appear in a corporate PPT, rewrite it in spoken Chinese.

## Example

Weak:

`未来更值钱的是你能不能把 AI 变成一个可交付的结果。`

Better:

`未来更值钱的是，你能不能真的搞出一个 AI Agent，让一家电商店每天自动整理客户问题、自动生成回复、自动统计哪些商品被问得最多，老板第二天能直接拿着这张表开会。`
